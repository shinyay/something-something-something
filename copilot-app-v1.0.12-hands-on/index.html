試したい機能を選ぶChoose a feature to try

15 機能それぞれに、前提・ステップ・確認ポイント・ヒントをまとめた独立ページを用意しています。上から順に進めても、必要なものだけ選んでも構いません。Each of the 15 features has its own page with prerequisites, steps, what to check, and tips. Work through them in order, or jump to just the ones you need.

01

My Work（マイワーク）My Work

全リポジトリのセッション・Issue・PR・自動化を一望するダッシュボードを開き、次の作業を選ぶ。Open the dashboard that spans every repo's sessions, issues, PRs, and automations, and pick your next task.

体験するTry it
02

並列エージェント開発Parallel agent development

独立した Git worktree で複数セッションを同時に走らせ、衝突なく並行開発する。Run several sessions at once in isolated Git worktrees and develop in parallel without conflicts.

体験するTry it
03

クラウドセッションCloud sessions

GitHub ホストの隔離環境でエージェントを動かし、別デバイスから引き継ぐ。Run an agent in a GitHub-hosted isolated environment and pick it up from another device.

体験するTry it
04

Canvas（キャンバス）Canvas

人とエージェントが双方向に編集できる作業サーフェスで、プランや PR をその場で操作する。Use the two-way work surface where people and agents edit plans and PRs together, in place.

体験するTry it
05

Automations（自動化）Automations

繰り返しのプロンプトを、CRON でスケジュール実行する自動化に変える。Turn a repetitive prompt into a CRON-scheduled automation that runs on its own.

体験するTry it
06

Agent Merge

PR を CI 監視・レビュー対応・マージ条件充足までエージェントに任せる。Delegate a PR through CI watching, review responses, and merge conditions to the agent.

体験するTry it
07

クラウド / ローカル サンドボックスCloud / local sandboxes

本番に触れずにエージェントがコードを実行・検証できる境界環境を試す。Try the bounded environments where agents run and verify code without touching production.

体験するTry it
08

Copilot コードレビューCopilot code review

PR に Copilot レビューを依頼し、/security-review や /rubberduck を試す。Request a Copilot review on a PR and try /security-review and /rubberduck.

体験するTry it
09

Copilot SDK

アプリと同じエージェントランタイムを SDK から呼び、最初のスクリプトを動かす。Call the same agent runtime that powers the app from the SDK and run your first script.

体験するTry it
10

Copilot CLI

ターミナルから同じ基盤を使い、TUI・音声モード・/every を体験する。Use the same foundation from the terminal and try the TUI, voice mode, and /every.

体験するTry it
11

クラウドエージェントCloud agent

スケジュールや GitHub イベントで動き、承認を挟みながら Issue に対応させる。Have it run on a schedule or GitHub event and address issues while asking for approval.

体験するTry it
12

Memory++ と /chronicleMemory++ and /chronicle

アプリ・CLI・VS Code をまたいだ過去セッションの文脈を横断照会する。Query context from past sessions across the app, the CLI, and VS Code.

体験するTry it
13

パートナーエージェントアプリPartner agent apps

サードパーティのエージェントアプリを接続し、Issue を個別エージェントにアサインする。Connect third-party agent apps and assign issues to individual agents.

体験するTry it
14

拡張性（MCP / Skills / Plugins）Extensibility (MCP / Skills / Plugins)

MCP サーバー接続・カスタムスキル・プラグインで、自分の文脈をエージェントに持ち込む。Bring your own context into agents with MCP servers, custom skills, and plugins.

体験するTry it
15

Chats（旧 Quick Chats）Chats (formerly Quick Chats)

リポジトリを選ばずにチャットを開始し、ツール承認コマンドやフォークを試す。Start a chat without picking a repo, and try tool-approval commands and forking.

体験するTry it
