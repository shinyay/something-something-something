My Work（マイワーク）を体験するExperience My Work

接続したすべてのリポジトリを横断し、進行中のセッション・Issue・プルリクエスト・自動化を一つのビューで俯瞰する「司令塔」。次に何をやるべきかを、ここから決めます。The command center that spans every connected repository — in-progress sessions, issues, pull requests, and automations in a single view. This is where you decide what to do next.

この機能でできることWhat you'll experience

体験のゴールYour goal

このガイドを終えると、My Work ダッシュボードを開いて全リポジトリの状況を俯瞰し、そこから直接、次の作業に着手できるようになります。By the end of this guide you'll be able to open the My Work dashboard, get a cross-repository overview, and start your next task directly from it.

横断的な俯瞰A cross-cutting overview

複数リポジトリのセッション・Issue・PR・自動化を 1 画面で確認。See sessions, issues, PRs, and automations from multiple repos on one screen.

次の一手の判断Decide the next move

レビュー待ち・失敗中・停止中の作業を見分け、優先順位を付ける。Spot what's awaiting review, failing, or paused, and prioritize.

その場で着手Start in place

カードから直接セッションを開いたり、Issue をエージェントに渡したりできる。Open a session or hand an issue to an agent straight from a card.

前提Prerequisites

始める前にBefore you begin

用意するものWhat you need

  • GitHub Copilot App v1.0.12 をインストール済み（Windows / macOS / Linux）GitHub Copilot App v1.0.12 installed (Windows / macOS / Linux)
  • 有効な有料 Copilot サブスクリプション（Pro / Pro+ / Max / Business / Enterprise）でサインイン済みSigned in with an active paid Copilot subscription (Pro / Pro+ / Max / Business / Enterprise)
  • 少なくとも 1 つの GitHub リポジトリへのアクセス（Issue や PR がいくつかあると体験しやすい）Access to at least one GitHub repository (a few issues or PRs make it easier to experience)
  • ローカルでコードを動かす場合は Git がインストール済みGit installed if you plan to run code locally
ハンズオン手順Step-by-step

My Work を実際に使ってみるPut My Work to work

上から順に進めてください。各ステップは数分で完了します。Follow the steps in order — each takes just a few minutes.

  1. セットアップSetup

    アプリを開いてサインインするOpen the app and sign in

    Copilot App を起動し、GitHub アカウントでサインインします。初回は接続したいリポジトリやオーガニゼーションへのアクセスを許可します。Launch the Copilot App and sign in with your GitHub account. On first run, grant access to the repositories or organizations you want to connect.

  2. 接続Connect

    リポジトリを 2 つ以上つなぐConnect two or more repositories

    「横断ビュー」の価値を体験するため、複数のリポジトリを接続します。My Work は接続したリポジトリすべてを対象に情報を集約します。Connect multiple repositories so you can experience the cross-cutting view. My Work aggregates across every repository you connect.

    • 日常的に使うものと、Issue / PR がいくつかあるものを混ぜると違いが見えやすい。Mixing an everyday repo with one that has a few issues / PRs makes the difference clearer.
  3. 俯瞰Overview

    My Work ダッシュボードを開くOpen the My Work dashboard

    サイドバーから My Work を開きます。進行中のセッション、割り当てられた Issue、自分の PR、バックグラウンドの自動化が、リポジトリを横断して 1 画面にまとまって表示されます。Open My Work from the sidebar. In-progress sessions, assigned issues, your PRs, and background automations are gathered onto one screen, across repositories.

  4. 読み解くRead

    状態バッジで「今どうなっているか」を掴むUse status to grasp the current state

    各カードの状態（実行中・レビュー待ち・CI 失敗・停止など）を見て、注意が必要な作業を特定します。レビュー待ちや失敗中のものは、次に手を付ける有力候補です。Scan each card's status (running, awaiting review, CI failing, paused, etc.) to identify work that needs attention. Anything awaiting review or failing is a strong candidate for your next move.

  5. 絞り込むFilter

    検索・フィルタで対象を絞るFilter and search to narrow down

    リポジトリ・種別（セッション / Issue / PR / 自動化）・状態で絞り込み、今のコンテキストに関係するものだけを表示します。数が多いときほど効果的です。Filter by repository, type (session / issue / PR / automation), or status to show only what's relevant right now. The more items you have, the more this helps.

  6. 着手Act

    カードから次の作業を始めるStart your next task from a card

    気になるカードを選び、その場で作業を開始します。例えば Issue を選んでエージェントセッションを起動したり、レビュー待ちの PR を開いて対応したり。My Work から離れずに着手できます。Pick a card and start right there — for example, launch an agent session from an issue, or open a PR that's awaiting review. You act without leaving My Work.

    • Issue → セッション起動の流れは、ガイド 02（並列エージェント開発）に続きます。The issue → session flow continues in guide 02 (parallel agent development).
確認ポイントWhat you should see

うまくいったサインSigns it worked

チェックリストChecklist

  • 複数リポジトリの項目が 1 つの My Work ビューに混在して表示されている。Items from multiple repositories appear together in a single My Work view.
  • 各項目に状態（実行中・レビュー待ち・失敗など）が表示され、区別できる。Each item shows a status (running, awaiting review, failing, etc.) you can distinguish.
  • フィルタを変えると表示件数が変化する。Changing a filter changes how many items are shown.
  • カードから、画面遷移なしにセッションや PR を開けた。You could open a session or PR from a card without a full page switch.
ヒント・応用Tips & variations

もう一歩使いこなすGo a little further

試してみるThings to try

  • 朝いちばんに My Work を開き、「レビュー待ち」だけをフィルタして、その日の順番を決める習慣にする。Open My Work first thing, filter to just “awaiting review,” and use it to plan the day's order.
  • 自動化（Automations）の行に注目し、バックグラウンドで走っているタスクの結果を確認する。→ ガイド 05 へ。Watch the automations row and check the results of tasks running in the background → see guide 05.
  • クラウドセッションの項目は別デバイスからも同じ状態で見える。→ ガイド 03 で引き継ぎを試す。Cloud session items appear with the same state on other devices → try handing one off in guide 03.
